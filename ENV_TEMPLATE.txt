# Git에는 절대 올리지 마세요!

# ========================================
# 솔라피 SMS API 설정
# ========================================
# 솔라피 대시보드에서 확인 가능
# https://console.solapi.com/

NEXT_PUBLIC_SOLAPI_API_KEY=your_api_key_here
NEXT_PUBLIC_SOLAPI_API_SECRET=your_api_secret_here
NEXT_PUBLIC_SOLAPI_PFID=your_pfid_here
NEXT_PUBLIC_SENDER_PHONE=your_sender_phone_here

# ========================================
# 설정 방법
# ========================================
# 1. 솔라피 회원가입 (https://solapi.com)
# 2. 대시보드 로그인
# 3. API Keys 메뉴에서 키 생성
# 4. 발신번호 등록 (사전 등록 필수)
# 5. 위 값들을 실제 값으로 교체

# ========================================
# 주의사항
# ========================================
# - NEXT_PUBLIC_ 접두사 유지 필수
# - 발신번호는 하이픈(-) 없이 입력
# - 예: 01012345678 (O), 010-1234-5678 (X)
# - API Key와 Secret은 외부 노출 금지

# ========================================
# 테스트 방법
# ========================================
# 1. 개발 서버 실행 (pnpm dev)
# 2. 문의하기 폼 테스트
# 3. 솔라피 대시보드에서 발송 내역 확인

# ========================================
# Vercel 배포 시
# ========================================
# 1. Vercel 대시보드 → Settings
# 2. Environment Variables 메뉴
# 3. 위 4개 변수를 모두 추가
# 4. 재배포 실행
