반드시 지켜야 할 사항:

1. 컴포넌트 구조 (그대로 사용)
   - Header.tsx: 드롭다운 네비게이션 구조 유지
   - Footer.tsx: 연락처 CTA + 카카오톡 버튼 구조 유지
   - /ui 폴더: shadcn 컴포넌트 그대로 사용

2. 페이지 구조 패턴
   - 모든 페이지에 gradient-bg 배경 적용
   - container mx-auto max-w-6xl 레이아웃 유지
   - 히어로 섹션 → 콘텐츠 섹션 구조

3. 이미지 표시 표준
   - 중앙 정렬: display: flex, justifyContent: center
   - Next.js Image 컴포넌트 사용
   - width={1600} height={1200} 기본값

4. 스타일링 방식
   - Tailwind CSS + styled-jsx 혼용 유지
   - 기존 gradient 색상 패턴 유지
   - 반응형 디자인 패턴 유지

5. 기능 구현
   - SMS 발송 기능 (솔라피 API)
   - 모바일 전화 연결 버튼
   - 카카오톡 상담 버튼
   - 네비게이션 드롭다운 메뉴

6. 변경할 내용
   - 회사 정보 (이름, 전화번호, 주소)만 변경
   - 이미지 경로와 alt 텍스트
   - 페이지 콘텐츠 텍스트

타겟 사이트의 이미지는 다운로드해서 /public 폴더에 저장하고,
동일한 레이아웃과 구조로 구현해줘."
